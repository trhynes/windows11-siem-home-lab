# Lab Environment

Host Machine:
- MacBook Pro 2020 M1
- macOS

Virtualization:
- UTM

- Windows 11 Pro

Lab Purpose:
- Security monitoring
- Event log analysis
- Vulnerability management

Build a beginner SOC-focused home lab to gain hands-on experience with Windows logging and SIEM technologies.
